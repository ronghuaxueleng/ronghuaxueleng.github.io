Boolenзұ»

еңЁECMAScriptдёӯеҫҲе°‘дҪҝз”ЁBooleanеҜ№иұЎпјҢеӣ дёәдёҚжҳ“зҗҶи§ЈпјҢеҰӮпјҡ

еңЁиҝҷж®өд»Јз ҒдёӯпјҢз”ЁfalseеҖјеҲӣе»әBooleanеҜ№иұЎпјҢз„¶еҗҺз”ЁиҝҷдёӘеҖјдёҺеҺҹе§ӢеҖјtrueиҝӣиЎҢ AND ж“ҚдҪңгҖӮеңЁBooleanиҝҗз®—дёӯпјҢfalseе’ҢtrueиҝӣиЎҢAND ж“ҚдҪңзҡ„з»“жһңжҳҜ falseгҖӮдёҚиҝҮпјҢеңЁиҝҷиЎҢд»Јз ҒдёӯпјҢи®Ўз®—зҡ„жҳҜoFalseObjectпјҢиҖҢдёҚжҳҜе®ғзҡ„еҖјfalseгҖӮеңЁBooleanиЎЁиҫҫејҸдёӯпјҢжүҖжңүеҜ№иұЎйғҪдјҡиў«иҮӘеҠЁиҪ¬жҚўдёәtrueпјҢжүҖд»Ҙз»“жһңдёәtrueгҖӮеҸӮиҖғдёӢйқўзҡ„д»Јз Ғпјҡ

Numberзұ»

д»ҘдёҠдёүдёӘж–№жі•йғҪдјҡиҝӣиЎҢиҲҚе…Ҙж“ҚдҪңгҖӮзӨәдҫӢд»Јз Ғпјҡ

Stringзұ»

Stringзұ»е…·жңүlengthеұһжҖ§пјҢе®ғжҳҜеӯ—з¬ҰдёІдёӯзҡ„еӯ—з¬ҰдёӘж•°пјҢеҸҢеӯ—иҠӮеӯ—з¬Ұд№ҹз®—дёҖдёӘеӯ—з¬ҰгҖӮ

Stringзұ»жңүеӨ§йҮҸзҡ„ж–№жі•пјҢдё»иҰҒд»Ӣз»ҚеҰӮдёӢпјҡ

charAt(ж•ҙеһӢеҸӮж•°)пјҡиҝ”еӣһеӯ—з¬ҰдёІдёӯеҚ•дёӘеӯ—з¬ҰгҖӮзӨәдҫӢпјҡ

charCodeAt(ж•ҙеһӢеҸӮж•°)пјҡиҝ”еӣһеӯ—з¬ҰдёІдёӯеҚ•дёӘеӯ—з¬Ұд»Јз ҒгҖӮзӨәдҫӢпјҡ

concat(еӯ—з¬ҰдёІ)пјҡжҠҠдёҖдёӘжҲ–еӨҡдёӘеӯ—з¬ҰдёІиҝһжҺҘеҲ°StringеҜ№иұЎзҡ„еҺҹе§ӢеҖјдёҠгҖӮзӨәдҫӢпјҡ

indexOf(еӯ—з¬ҰдёІ)пјҡиҝ”еӣһжҢҮе®ҡзҡ„еӯ—з¬ҰдёІеңЁеҸҰдёҖдёӘеӯ—з¬ҰдёІдёӯзҡ„дҪҚзҪ®пјҲд»Һеӯ—з¬ҰдёІзҡ„ејҖеӨҙиҝӣиЎҢжЈҖзҙўпјүгҖӮ

lastIndexOf(еӯ—з¬ҰдёІ)пјҡиҝ”еӣһжҢҮе®ҡзҡ„еӯ—з¬ҰдёІеңЁеҸҰдёҖдёӘеӯ—з¬ҰдёІдёӯзҡ„дҪҚзҪ®пјҲд»Һеӯ—з¬ҰдёІзҡ„з»“е°ҫиҝӣиЎҢжЈҖзҙўпјүгҖӮзӨәдҫӢпјҡ

localeCompare(еӯ—з¬ҰдёІ)пјҡеҜ№еӯ—з¬ҰдёІиҝӣиЎҢжҺ’еәҸпјҢиҝ”еӣһеҖјжҳҜдёӢеҲ—дёүдёӘд№ӢдёҖпјҡ

AгҖҒеҰӮжһңStringеҜ№иұЎжҢүз…§еӯ—жҜҚйЎәеәҸжҺ’еңЁеҸӮж•°дёӯзҡ„еӯ—з¬ҰдёІд№ӢеүҚпјҢиҝ”еӣһиҙҹж•°(йҖҡеёёжҳҜ-1пјҢдҪҶзңҹжӯЈиҝ”еӣһеҖјз”ұе…·дҪ“е®һзҺ°еҶіе®ҡ)

BгҖҒеҰӮжһңStringеҜ№иұЎзӯүдәҺеҸӮж•°дёӯзҡ„еӯ—з¬ҰдёІпјҢиҝ”еӣһ0

CгҖҒеҰӮжһңStringеҜ№иұЎжҢүз…§еӯ—жҜҚйЎәеәҸжҺ’еңЁеҸӮж•°дёӯзҡ„еӯ—з¬ҰдёІд№ӢеҗҺпјҢиҝ”еӣһжӯЈж•°(йҖҡеёёжҳҜ1пјҢдҪҶзңҹжӯЈиҝ”еӣһеҖјз”ұе…·дҪ“е®һзҺ°еҶіе®ҡ)

зӨәдҫӢпјҡ

slice(ж•ҙеһӢеҸӮж•°[,ж•ҙеһӢеҸӮж•°])гҖҒsubstring(ж•ҙеһӢеҸӮж•°[,ж•ҙеһӢеҸӮж•°])пјҡд»ҺеӯҗдёІеҲӣе»әеӯ—з¬ҰдёІеҖјгҖӮз¬¬дёҖдёӘеҸӮж•°жҳҜиҰҒиҺ·еҸ–зҡ„еӯҗдёІзҡ„иө·е§ӢдҪҚзҪ®пјҢз¬¬дәҢдёӘеҸӮж•°жҳҜиҰҒиҺ·еҸ–зҡ„еӯҗдёІз»ҲжӯўеүҚзҡ„дҪҚзҪ®пјҢеҰӮжһңзңҒз•Ҙз¬¬дәҢеҸӮж•°пјҢз»ҲжӯўдҪҚе°ұй»ҳи®Өдёәеӯ—з¬ҰдёІй•ҝеәҰгҖӮиҝҷдёӨдёӘж–№жі•йғҪдёҚж”№еҸҳStringеҜ№иұЎиҮӘиә«еҖјгҖӮеҪ“еҸӮж•°дёәжӯЈж—¶дёӨдёӘж–№жі•з”Ёжі•еҸҠиҝ”еӣһеҖјеқҮзӣёеҗҢпјҢеҸӘжңүеҸӮж•°жңүиҙҹеҖјж—¶жүҚдёҚеҗҢгҖӮеҜ№дәҺиҙҹеҸӮж•°пјҢslice()ж–№жі•дјҡз”Ёеӯ—з¬ҰдёІзҡ„й•ҝеәҰеҠ дёҠеҸӮж•°пјҢsubstring()е°Ҷе…¶еҒҡдёә0еӨ„зҗҶгҖӮеҸҰеӨ–еҰӮжһңжңүдёӨдёӘеҸӮж•°пјҢз¬¬дәҢдёӘжҜ”з¬¬дёҖдёӘе°Ҹж—¶пјҢslice()иҝ”еӣһзҡ„еҖјдёәз©әпјҢsubstring()дјҡжҠҠиҫғе°Ҹзҡ„дҪңдёәз¬¬дёҖдёӘеҸӮж•°гҖӮдёәзӨәдҫӢпјҡ

toLowerCase()гҖҒtoLocaleLowerCase()гҖҒtoUpperCase()гҖҒtoLocaleUpperCase()пјҡеүҚдёӨдёӘз”ЁдәҺжҠҠеӯ—з¬ҰдёІиҪ¬жҚўдёәе…Ёе°ҸеҶҷпјҢеҗҺдёӨдёӘз”ЁдәҺжҠҠеӯ—з¬ҰдёІиҪ¬жҚўдёәе…ЁеӨ§еҶҷгҖӮtoLowerCase()и·ҹtoUpperCase()жҳҜеҺҹе§Ӣж–№жі•пјҢtoLocaleLowerCase()и·ҹtoLocaleUpperCase()жҳҜеҹәдәҺзү№е®ҡеҢәеҹҹе®һзҺ°зҡ„гҖӮзӨәдҫӢпјҡ
